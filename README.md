## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:

- Você instalou a versão mais recente de `Java 21 / O sistema instala dependencias automaticamente com o pom.xml`
- Você tem uma máquina `Windows / Linux / Mac`. Tendo JDK 21 instalado o sistema vai rodar.

## ☕ Usando **reserva-de-salas-API**

Você pode rodar a aplicação **de duas formas**:

# 1️⃣ Rodando localmente (sem Docker)

---

### 🐧 **No Linux/macOS**

```bash
git clone https://github.com/jao761/reserva-de-salas-API.git
cd reserva-de-salas-API
./mvnw spring-boot:run
```
### 🪟 **No Windows (PowerShell ou CMD)**

```bash
git clone https://github.com/jao761/reserva-de-salas-API.git
cd reserva-de-salas-API
./mvnw spring-boot:run
```

## 🐳 Executando com Docker Compose

Este projeto já acompanha um **Dockerfile**, portanto a execução com containers é simples.

### ▶️ 1. Subir a aplicação

No diretório raiz do projeto, execute:

```bash
docker compose up -d --build
```
